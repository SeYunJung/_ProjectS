# Tower Defense Project

## 🎮 게임 소개
**『Tower Defense』**는 C#, Unity로 구현했던 스파르타 던전 일부를 유니티로 만든 게임입니다.

Unity 엔진 기반으로 개발되었으며, Singleton, Object Pool을 활용하여 구현하였습니다.

- 장르: 디펜스

- 플랫폼: PC (Windows)

- 개발 툴: Unity 2022.3.62f3, Visual Studio, GitHub

- 개발 기간: 2025.11.17 ~ 2025.11.24

## 🕹️ 플레이 방법

- 빈 블록을 클릭해 재화(돈)를 소모하고 타워를 배치합니다.
- 같은 레벨 타워가 둘 이상있으면 승급할 수 있습니다.
- 레벨4 영웅이 둘 이상있으면 재화(미네랄)을 소모하고 초월할 수 있습니다.
- 초록색 블록에 타워를 배치하면 공격 속도가 증가합니다.
- 탐사정으로 재화(미네랄)을 획득할 수 있습니다. (탐사정은 돈으로 구입 가능)
- Mission, Upgrade는 미구현입니다. 


## 🛠 사용 기술
- Unity 2022.3.62f3
- C# (게임 로직 및 시스템 구현)
- GitHub (버전 관리)
- TextMeshPro (UI 시스템 구성)
- 단일 Singleton (전역 접근)
- Object Pool (투사체 생성)


## 🌟 주요 구현 기능 
 - 전역적으로 필요한 객체는 단일 Singleton으로 구현
 - 투사체를 Object Pool로 관리 


## ✨ 구현 기능
1. 적 생성, 적 이동 
- 적 생성 : SpawnManger
- 적 이동 : 좌표 이동 (transform.position)

2. 영웅 공격
- 투사체 생성 : Object Pool
- 공격 : 투사체와 목표 몬스터의 거리를 계산해 공격처리 

3. 적 체력 감소, 적 사망처리 (보스X)
- 적 체력 감소 : 투사체와 목표 몬스터 거리가 가까우면 적 체력 감소
- 적 사망처리 : 적 체력이 0이하일 때 사망처리

4. 웨이브 승리/실패 판정
- 웨이브 승리 : 웨이브 몬스터를 모두 처치
- 웨이브 실패 : 플레이어 체력이 0이하일 때 

5. 영웅 소환
- 재화(돈)을 소모해 영웅을 소환

6. 영웅 승급 
- 같은 레벨, 같은 종류 캐릭터가 둘 이상일 경우 영웅 승급

7. 영웅 초월
- 같은 레벨, 같은 종류 캐릭터가 둘 이상일 경우 재화(미네랄)을 소모해 영웅 초월

8. 탐사정
- 재화(미네랄)을 수확하는 오브젝트 

9. 수리
- 재화(미네랄)을 소모해 막힌 막힌 블록을 빈 블록으로 전환

10. 타일 버프 
- 초록색 타일에 놓인 영웅은 공격 속도가 증가 

## 🧠트러블 슈팅  
<p><b>투사체 딜레이가 적용되지 않는 현상</b></p>
<p>3초마다 투사체가 생성되고 발사될 수 있게 만들었으나, 3초 딜레이가 적용되지 않음</p>
<p><b>원인</b></p>
<p>투사체가 3초 딜레이가 적용되기 전에 오브젝트 풀에서 재활용되어 딜레이가 적용되지 않음.</p>
<p><b>해결</b></p>
<p>Update에서 공격 상태 처리를 하니 투사체가 오브젝트 풀에서 계속 재활용이 됨. 공격을 해야할 시점에 공격처리를 하도록 수정하니 해결됨.</p>
